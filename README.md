## Installing

Select the module and import it through GitHub.

```yaml
dependencies:
  flutter:
    sdk: flutter
  core_resources:
    git:
      url: https://github.com/icarohs7/unox-flutter-resmodules
      path: core_resources
```

Note: No internationalization, all strings are in pt-BR
